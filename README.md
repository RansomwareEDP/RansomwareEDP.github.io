# Ransomware Ecosystem Network Observatory

Homepage for the Ransomware Ecosystem Network Observatory, a structural analysis of Russia and CIS-linked ransomware networks and the financial, infrastructure, and jurisdictional leverage points that can degrade them.

**Live site:** https://ransomwareedp.github.io/

This repository holds the landing page. The full research site (ecosystem map, group profiles, infrastructure profiles, the KRYSHA simulator, the live intelligence feed, and the document library) lives in the companion repository and is served at https://ransomwareedp.github.io/Mapping/.

Open research, maintained by Reno.
